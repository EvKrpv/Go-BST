# Go-BST
Реализация обобщённого двоичного дерева Binary Search Tree

Описание
Программа реализует классическое бинарное дерево поиска с следующими возможностями:

  - Вставка элементов (ключ-значение)
  - Поиск значений по ключу
  - Удаление элементов по ключу
  - Обход в порядке возрастания (in-order traversal)

Использование

Программа работает в интерактивном режиме, принимая команды из стандартного ввода:

Доступные команды:
```
Insert <key> <value>    - Добавить элемент в дерево
Find <key>              - Найти значение по ключу
Delete <key>            - Удалить элемент по ключу
InOrder                 - Вывести все элементы в отсортированном порядке
```

Пример использования:
```
Insert 5 apple
Insert 3 banana
Insert 7 orange
Find 3           # Выведет: banana
Find 10          # Выведет: not found
InOrder          # Выведет: 3: banana
                 #          5: apple
                 #          7: orange
Delete 5
InOrder          # Выведет: 3: banana
                 #          7: orange
```

Особенности реализации
  - Обобщённый код (generics): Работает с любыми сравниваемыми типами ключей и любыми типами значений
  - Рекурсивные алгоритмы: Все операции реализованы через рекурсию
  - Эффективный обход: Используется итератор для обхода дерева

Структура данных
```
type Node[K cmp.Ordered, V any] struct {
    Key   K
    Value V
    Left  *Node[K, V]
    Right *Node[K, V]
}

type BST[K cmp.Ordered, V any] struct {
    Root *Node[K, V]
}
```
Запуск
```
go run main.go
```
Затем вводите команды в консоль. Для завершения работы используйте Ctrl+D (Linux/Mac) или Ctrl+Z (Windows).
